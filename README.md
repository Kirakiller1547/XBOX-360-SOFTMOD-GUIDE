# Xbox 360 Softmod Guide (BadAvatar / Aurora)

A guide for softmodding an Xbox 360, loading games digitally, applying custom themes/skins, and doing a case mod.

## 1. Requirements

You'll need a USB drive or another storage drive to set up BadAvatar and store games. An external SSD, or a normal SSD paired with a USB-to-SATA adapter, works great.

**My setup:**
- 128GB SSD + USB-SATA adapter → running BadAvatar
- 1TB external HDD → plugged into the back of the console
- Internal 120GB Xbox 360 drive

**Recommended hardware:**
- USB drive: [Kingston DataTraveler Exodia 64GB USB 3.2](https://www.newegg.com/kingston-technology-corp-datatraveler-exodia-64gb-usb-3-2-gen-1-black-blue/p/0BD-00CA-00D98)
- SSD: [Intenso Top Performance 128GB](https://geizhals.de/intenso-top-performance-ssd-128gb-3812430-a881749.html)
- HDD: [Seagate Expansion Portable 1TB](https://geizhals.de/seagate-expansion-portable-stkm-1tb-stkm1000400-a2558781.html)

### Steps
1. Plug the USB drive or external SSD into a USB port on the Xbox 360.
2. Format the drive under **Storage Devices** in the console settings.
3. Disable **auto sign-in** on your Xbox profiles.
4. Follow this video guide by ChaboiFam to install BadAvatar: [YouTube Guide](https://www.youtube.com/watch?v=z7Fbi8zN8tw)

## 2. Adding Games

1. Search for "Xbox 360 ROMs" and download the ISO files from a site of your choice.
2. Use [iso2god](https://iso2god.de.softonic.com/) to convert the ISO into a GOD (Games on Demand) image.
3. Copy the converted files onto a USB device, plug it into the Xbox, and move the files to your local disk — or just leave them on the external hard drive. Location doesn't matter, as long as you point Aurora to the correct path.
4. To download game covers/artwork: press **Start** on your controller (make sure you're connected to the internet), then select **Update Assets**.

## 3. Themes & Skins

For this project I used the [Xbox Series X Skin](https://www.realmodscene.com/index.php?/topic/25139-xbox-series-skin/) from RealModScene.

Since the original download links are no longer available, the files are included in this repo's **Releases** section.

1. Follow the installation guide on the skin's page.
2. Place all files in the correct folders on your BadAvatar drive.
3. On the console, press **B** → go to **Skins** → select the Xbox Series skin.
4. Under **Theme**, disable **Animation**.

## 4. Game Updates via Xbox Unity

1. Register and activate your account at [xboxunity.net](https://xboxunity.net/) (confirm via email).
2. On the console, go to **Settings → Profile** and enable **Unity Connection**.
3. Highlight any game and press **Y** → select the download source/controller.
4. Navigate right to see and install all available updates for that game.

## 5. Moving Physical Discs to Digital

To rip your physical game discs onto the Xbox 360 as digital copies, follow this guide: [YouTube Guide](https://www.youtube.com/watch?v=blxinVOkLLc&t=182s)
