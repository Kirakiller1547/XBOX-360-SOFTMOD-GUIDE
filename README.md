## 1. 📦 Requirements

You'll need a USB drive or another storage drive to set up **BadAvatar** and store games. An external SSD, or a normal SSD paired with a USB-to-SATA adapter, works great.

### My Setup

| Component | Details |
|---|---|
| 🔹 Boot / Games drive | 128GB SSD + USB-SATA adapter, running BadAvatar |
| 🔹 Extra storage | 1TB external HDD, plugged into the back of the console |
| 🔹 Internal drive | 120GB stock Xbox 360 drive |

### Recommended Hardware

| Part | Link |
|---|---|
| USB drive | [Kingston DataTraveler Exodia 64GB USB 3.2](https://www.newegg.com/kingston-technology-corp-datatraveler-exodia-64gb-usb-3-2-gen-1-black-blue/p/0BD-00CA-00D98) |
| SSD | [Intenso Top Performance 128GB](https://geizhals.de/intenso-top-performance-ssd-128gb-3812430-a881749.html) |
| HDD | [Seagate Expansion Portable 1TB](https://geizhals.de/seagate-expansion-portable-stkm-1tb-stkm1000400-a2558781.html) |

### Steps

- [ ] Plug the USB drive or external SSD into a USB port on the Xbox 360.
- [ ] Format the drive under **Storage Devices** in the console settings.
- [ ] Disable **auto sign-in** on your Xbox profiles.
- [ ] Follow this video guide by ChaboiFam to install BadAvatar → [YouTube Guide](https://www.youtube.com/watch?v=z7Fbi8zN8tw)

## 2. 🎮 Adding Games

- [ ] Search for "Xbox 360 ROMs" and download the ISO files from a site of your choice.
- [ ] Use [iso2god](https://iso2god.de.softonic.com/) to convert the ISO into a GOD (Games on Demand) image.
- [ ] Copy the converted files onto a USB device, plug it into the Xbox, and move the files to your local disk — or just leave them on the external hard drive. Location doesn't matter, as long as you point Aurora to the correct path.
- [ ] To download game covers/artwork: press **Start** on your controller (make sure you're connected to the internet), then select **Update Assets**.

## 3. 🎨 Themes & Skins

For this project I used the **[Xbox Series X Skin](https://www.realmodscene.com/index.php?/topic/25139-xbox-series-skin/)** from RealModScene.

> **Note:** Since the original download links are no longer available, the files are included in this repo's [**Releases**](../../releases) section.

### Installation

- [ ] Follow the installation guide on the skin's page.
- [ ] Place all files in the correct folders on your BadAvatar drive.
- [ ] On the console, press **B** → go to **Skins** → select the Xbox Series skin.
- [ ] Under **Theme**, disable **Animation**.

## 4. 🔄 Game Updates via Xbox Unity

- [ ] Register and activate your account at [xboxunity.net](https://xboxunity.net/) (confirm via email).
- [ ] On the console, go to **Settings → Profile** and enable **Unity Connection**.
- [ ] Highlight any game and press **Y** → select the download source/controller.
- [ ] Navigate right to see and install all available updates for that game.

## 6. 🛠️ Case Mod

The full build: military-green top/front/bottom panels, black sides, an acrylic window over the disc drive, a repainted and "flexed" drive tray so the disc is visible while spinning, internal LED lighting, a 120mm intake fan mounted on the acrylic panel blowing onto the heatsink, and a custom drive bay for the external SSD/HDD — all tied together with matching green/black controllers.

### Parts List

| Part | Link |
|---|---|
| 120mm LED Fan (green) | [eBay](https://www.ebay.de/itm/157915753200) |
| Acrylic glass panel | [eBay](https://www.ebay.de/itm/167171910591) |
| Green spray paint | [eBay](https://www.ebay.de/itm/252335750703?var=551180485617) |
| Black spray paint | [eBay](https://www.ebay.de/itm/252335750703?var=551180485609) |
| Controller thumbsticks | [eBay](https://www.ebay.de/itm/186237759446) |
| LED Stripe (+ remote controller) | [eBay](https://www.ebay.de/itm/293694271309) |

### Build Overview

- [ ] Spray-paint front/top/bottom panels **military green**, side panels **black** (use a plastic primer first for better adhesion).
- [ ] Cut a window into the top panel and fit the **acrylic glass panel**.
- [ ] Repaint the disc drive faceplate and "flex" it into place so the spinning disc is visible through the window.
- [ ] Route the **LED stripe** along the acrylic window edges, powered from a 12V source inside the console (e.g. fan header), controlled via the included remote.
- [ ] Mount the **120mm fan** on top of the acrylic panel, aligned to blow onto the heatsink.
- [ ] Paint the fan grille/tray green.
- [ ] Build/paint a bracket behind the fan to hold the external SSD/HDD, and route their cables cleanly.
- [ ] Repaint the controllers to match (green/black).
